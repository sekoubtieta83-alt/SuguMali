# **App Name**: ConnectSphere

## Core Features:

- User Account Creation: Allow users to create accounts with email/password/phone number or social login.
- Media Upload: Enable users to upload photos and videos.
- Profile Verification: Admin panel to verify user profiles.
- Admin Dashboard: Admin interface to view app usage metrics (users, visits, etc.).
- Promote Products: Feature allowing users to promote products.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) for trust and reliability.
- Background color: Light gray (#F5F5F5) to ensure readability and a clean look.
- Accent color: Vibrant orange (#FF9800) to draw attention to important actions.
- Body and headline font: 'Inter', a versatile sans-serif, used for both headlines and body text.
- Modern, minimalist icons to represent various actions and categories.
- Clean, intuitive layout for easy navigation and content discovery.
- Subtle transitions and animations for a smooth user experience.
